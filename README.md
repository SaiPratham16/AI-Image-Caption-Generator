# AI Image Caption Generator

A web application that generates captions for uploaded images.

## 🚀 Features
- Upload image
- Generate caption
- Simple UI

## 🛠 Tech Stack
- Python
- Flask
- HTML, CSS, JavaScript

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   python app.py

3. Open:
   http://127.0.0.1:5000/
